# Datasets

This directory contains the smaller datasets used by the project:

- `BUSI-combined`
- `ClinicDB`
- `GlaS`
- `Kvasir-SEG`

## ISIC18_exp1

`ISIC18_exp1` is not included in this repository because the local processed
copy is approximately 13 GB, which is too large for normal GitHub repository
storage.

Please download the original HAM10000 / ISIC 2018 data from the official
sources and preprocess it locally if needed:

- HAM10000 paper DOI: https://doi.org/10.1038/sdata.2018.161
- HAM10000 dataset DOI: https://doi.org/10.7910/DVN/DBW86T
- ISIC Challenge datasets: https://challenge.isic-archive.com/data/#2018

Citation:

Tschandl, P., Rosendahl, C., and Kittler, H. The HAM10000 dataset, a large
collection of multi-source dermatoscopic images of common pigmented skin
lesions. Scientific Data 5, 180161 (2018).
https://doi.org/10.1038/sdata.2018.161
